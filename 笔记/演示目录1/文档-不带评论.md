# 示例文档
这是一个示例文档，用于展示所有的 markdown 功能。
```
---
<!-- 这里是标题内容-->
title: "{{title}}"
<!-- 这里是日期内容 -->
date: "{{date}}"
<!-- slug -->
slug: blog-post-slug
<!-- 这是文档的标签 -->
tags:
  - 标签
<!-- 这是文档的分类 -->
categories:
  - 分类
<!-- 这是文档的描述 -->
description: 描述
<!-- 这是文档的摘要 -->
summary: 摘要
<!-- 是否是草稿 -->
draft: false
<!-- 是否开启评论 -->
comment: false
---
```
